
# 💼 Personal Portfolio Website

## 📌 Overview

This is my personal portfolio website built to showcase my skills, projects, education, achievements, and contact information. The website provides a professional online presence for recruiters, collaborators, and potential employers.

## ✨ Features

* Responsive Design
* About Me Section
* Skills Showcase
* Projects Gallery
* Education Details
* Resume Download
* Contact Information
* Social Media Links
* Smooth Navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

```
Portfolio/

```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
